# PongHauKi (Arduboy)

A simple chess known in China and a few other Asian countries. Move 1 of 2 pieces by turn. Who gets blocked loses game.

[Pong Hau K'i - Wiki](https://en.wikipedia.org/wiki/Pong_Hau_K%27i)
